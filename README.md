# 基金看板 · Fund Dashboard

> **知识产权声明：开发来自林甲酸**  
> © 林甲酸。保留所有权利。

一个单文件基金看板网页，用于查看基金实时估值、历史净值、前十大重仓股和个股实时行情。页面支持添加、删除自选基金，并会把个人配置保存在当前浏览器中。

## 在线访问

部署到 GitHub Pages 后，可通过类似下面的链接访问：

```text
https://coslin4949.github.io/fund-dashboard/
```

> 如果当前链接暂时无法访问，请确认 GitHub Pages 是否已经开启，并且仓库根目录中存在 `index.html` 文件。

## 功能特性

- 基金实时估值展示
- 基金历史净值走势图
- 前十大重仓股展示
- 重仓股票实时价格与涨跌幅展示
- 支持添加 6 位基金代码
- 支持删除看板中的基金
- 默认基金删除后刷新仍保持隐藏
- 自选基金保存在浏览器 `localStorage` 中
- 暗色终端风格界面
- 页面左上角带有林甲酸知识产权署名

## 使用方式

### 本地打开

直接双击打开：

```text
fund_dashboard.html
```

如果要部署为 GitHub Pages，建议将页面文件命名为：

```text
index.html
```

然后上传到 GitHub 仓库根目录。

### 添加基金

1. 在页面右上角输入 6 位基金代码。
2. 点击 `添加`。
3. 添加成功后，新基金会出现在卡片列表中。
4. 刷新页面后，自选基金仍会保留在当前浏览器中。

### 删除基金

1. 点击基金卡片右上角的 `×`。
2. 在确认弹窗中确认删除。
3. 删除后该基金会从看板中移除。
4. 如果删除的是默认基金，刷新页面后仍然保持隐藏。

## 数据说明

本看板使用东方财富 / 天天基金公开接口获取数据，包括但不限于：

- 基金实时估值
- 历史净值
- 基金重仓股
- 股票实时行情

数据仅供学习、研究和个人参考使用，不构成任何投资建议。

## 浏览器存储说明

本项目没有后端服务器，也不会上传用户数据。

用户添加或删除的基金信息会保存在当前浏览器的 `localStorage` 中：

```text
fundDashboard.customFunds.v1
fundDashboard.hiddenDefaultCodes.v1
```

因此：

- 你在自己电脑上添加的基金，不会自动同步给别人。
- 别人打开同一个网页链接后，会使用他自己浏览器里的本地配置。
- 如果清理浏览器缓存或站点数据，自选基金配置可能会丢失。

## 部署到 GitHub Pages

1. 在 GitHub 新建仓库，例如：

   ```text
   fund-dashboard
   ```

2. 将页面文件改名为：

   ```text
   index.html
   ```

3. 将 `index.html` 和 `README.md` 上传到仓库根目录。

4. 进入仓库设置：

   ```text
   Settings → Pages
   ```

5. Source 选择：

   ```text
   Deploy from a branch
   ```

6. Branch 选择：

   ```text
   main / root
   ```

7. 保存后等待 GitHub 部署完成。

部署完成后，访问地址通常是：

```text
https://coslin4949.github.io/fund-dashboard/
```

## 注意事项

- 页面依赖外部公开接口，接口可能因网络、跨域策略或服务调整而暂时不可用。
- 部分 QDII / 联接基金可能没有完整的重仓股数据。
- 页面中的实时估值为公开接口返回的盘中估算值，不是最终基金净值。
- 投资有风险，数据仅供参考。
